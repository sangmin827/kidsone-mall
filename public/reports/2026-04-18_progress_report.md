# Kids One Mall 개발 진행 보고 (2026-04-18)

작성자: 이상민

---

## 1. 현재 구축 완료 상황

### 1-1. 기본 인프라
- Next.js 14+ App Router 기반 프로젝트 세팅
- Supabase 연동 (DB + Auth + Storage + RLS)
- 서버 컴포넌트 / 서버 액션 구조 적용
- 관리자 인증 게이트 (`requireAdmin`) 구축

### 1-2. 사용자 측 기능
- 회원 가입 / 로그인 / 로그아웃
- 상품 목록 페이지
- 카테고리별 상품 조회 페이지
- 상품 상세 페이지
- 장바구니 기본 기능 (담기 / 수량 변경 / 삭제)
- 결제창 기본 흐름 구축
- 결제 시 주문 정보 이동 로직
- 주문내역 페이지 연결

### 1-3. 관리자 (Admin) 페이지
- 카탈로그 통합 관리 페이지 (카테고리 = 폴더 / 상품 = 파일 트리 뷰)
- 상품 관리 (등록 / 수정 / 삭제 / 노출·숨김 토글)
- 상품 상세 모달 (빠른 수정)
- 카테고리 관리 (등록 / 수정 / 삭제 / 순서 이동 / 노출·숨김)
- 카테고리 RLS 정책 추가 (Admins INSERT / UPDATE / DELETE)
- 관리자 활동 로그(Admin Activity Log) 기록
- 빠른 편집 액션 (상품 이름/가격 인라인 수정, 카테고리 드롭다운 즉시 저장)

### 1-4. 장바구니 상품 개수 카운트
- 메인/헤더 장바구니 아이콘 옆에 담긴 상품 개수 배지 표시

## 2. 지금 꼭 메워야 하는 부분 (현재 비어있거나 부실)

### 2-1. 회원 관리 (현재 비약)
- 관리자 회원 목록 페이지 없음
- 회원 검색 / 필터 (가입일, 권한, 상태) 없음
- 회원 상세 (주문 이력, 누적 결제금액) 페이지 없음
- 권한 변경 (admin ↔ user) UI 없음
- 회원 차단 / 비활성화 기능 없음

### 2-2. 주문내역 관리 (현재 비약)
- 관리자 주문 목록 페이지 없음
- 주문 상태 변경 (결제완료 → 배송준비 → 발송 → 배송완료) 워크플로 없음
- 주문 상세 페이지 (배송지, 결제 정보, 상품 구성) 없음
- 주문 검색 / 필터 (기간, 상태, 회원) 없음
- 주문 취소 / 환불 처리 로직 없음
- CSV / Excel 주문 내보내기 없음

### 2-3. UI / UX
- 전체적인 디자인 꾸미기 작업 안 되어 있음 (기능 우선 구현 상태)
- 반응형 (모바일 / 태블릿) 확인 안 됨
- 로딩 상태 / 빈 상태(empty state) 꾸미기 부족
- 에러 페이지 (404, 500) 꾸미기 없음
- 접근성 (aria, 포커스 링, 키보드 네비게이션) 점검 안 됨

### 2-4. 사이트 테스트
- 전체 사용자 시나리오(가입 → 상품 탐색 → 장바구니 → 결제 → 주문확인) 끝단 테스트 안 됨
- 주요 엣지 케이스 (품절, 재고 0, 결제 실패, 중복 주문) 테스트 안 됨
- 브라우저 호환성 (Chrome / Safari / Edge / 모바일 브라우저) 확인 안 됨

### 2-5. 1:1 고객센터
- 문의하기 / 답변 시스템 없음
- 관리자 문의 목록 / 답변 UI 없음
- 문의 상태 (접수 / 처리중 / 완료) 관리 없음

### 2-6. 각 페이지 세부 조정
- 메인 페이지 영역 배치 / 썸네일 크기 등 세부 튜닝 필요
- 상품 상세 페이지 옵션 선택 UI 개선 필요
- 장바구니 / 결제 페이지 흐름 UX 다듬기 필요

## 3. 차후 구현 계획 (실제 쇼핑몰 수준을 위한 로드맵)

### 3-1. 우선순위 High (출시 직후 필수급)
- 세부 카테고리 확장 (2 depth 이상, 카테고리별 배너)
- 상품 리뷰 / 별점 시스템
- 찜(위시리스트) 기능
- 쿠폰 / 할인 코드 시스템
- 포인트 / 적립금 시스템
- 상품 재고 관리 (실재고 연동, 품절 자동 처리)
- 배송비 정책 (금액별 / 지역별)

### 3-2. 우선순위 Mid (차별화 기능)
- 이벤트 / 기획전 페이지
  - 사진 배너, 상세 이미지, 동영상 임베드
  - 기간 설정, 타임 세일
- 상품 검색 (키워드, 필터링, 자동완성)
- 추천 상품 (최근 본 상품, 함께 많이 구매한 상품)
- 상품 문의 (상품 상세 페이지에 Q&A)
- 알림톡 / 이메일 발송 (주문 확인, 배송 시작, 배송 완료)
- SNS 로그인 (카카오 / 네이버 / 구글)

### 3-3. 우선순위 Low (고도화)
- 관리자 대시보드 (매출 / 방문자 / 전환율 차트)
- 매출 통계 (기간별, 카테고리별, 상품별)
- 재방문 / 고객 세그먼트 분석
- A/B 테스트 프레임워크
- 다국어 / 다통화 지원
- PWA 지원 (모바일 앱 느낌)
- 라이브 커머스 / 실시간 채팅

### 3-4. 운영 / 인프라 측면
- 이미지 CDN 최적화 (썸네일 자동 생성, WebP)
- 검색엔진 최적화 (SEO 메타태그, sitemap.xml, robots.txt)
- 성능 모니터링 (Web Vitals, 에러 트래킹)
- 백업 / 복구 정책 수립
- 결제 모듈 이중화 (카드사 장애 대비)
- 이용약관 / 개인정보처리방침 / 환불규정 페이지 정비
